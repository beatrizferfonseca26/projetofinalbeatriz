Saloon — Beauty Appointment Management System

Scalable full stack web application designed for beauty salons and appointment-based businesses.  
Built with modern web technologies to manage scheduling, staff, services, inventory, and customer operations through a centralized platform.

Live Demo

Live Demo: https://projetofinalbeatriz.vercel.app/

Overview

Saloon is a full stack management platform focused on streamlining salon operations through appointment scheduling, service management, inventory control, and role-based administration.

The project was developed as a final course project and is currently evolving toward a scalable multi-tenant SaaS architecture with cloud database infrastructure and automated onboarding workflows.

Features

Appointment Management
- Create and manage appointments
- Reschedule and cancel bookings
- Staff availability control
- Appointment history tracking

Staff & Client Management
- Manage employees and schedules
- Client registration and management
- Role-based access control

Service & Inventory Management
- Service catalog management
- Product and inventory control
- Operational workflow support

Administration
- Full CRUD operations
- Availability configuration
- Administrative dashboard structure


User Roles

Client
- Create appointments
- Reschedule or cancel bookings
- View appointment history

Staff
- Manage appointments
- View schedules

Administrator
- Manage clients, staff, and services
- Manage products and inventory
- Configure staff availability
- Full system administration
